Tag Input

Bootstrap tag input is library to make it easy to input multiple item tag like value in some form input field.

About

This component visualize multiple items input in one form element ( textarea or input ) to make it easy to modify the json array value of the input.

Please note that the result of the input is always JSON array, and can only accept JSON array as preset value. The content that can't be JSON.parse will truncated and become empty array.
